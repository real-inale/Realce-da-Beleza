# Realce da Beleza

Website para aluguer de vestidos elegantes em Camama, Angola.

## Como executar

1. Instale as dependências:
   ```
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```
   npm run dev
   ```

3. Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Estrutura do Projeto

- `src/app/` - Páginas Next.js (Home, Vestidos, Acessórios, Sobre, Contacto)
- `src/components/` - Componentes reutilizáveis (Header, Footer)
- `tailwind.config.js` - Configuração do Tailwind CSS com cores e fontes personalizadas

## Funcionalidades

- Design responsivo com Tailwind CSS
- Integração com WhatsApp para reservas
- Páginas para catálogo de vestidos e acessórios
- Seções de sobre, contacto e depoimentos

## Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- Google Fonts (Playfair Display, Poppins)
