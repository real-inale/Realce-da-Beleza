Coloque aqui as imagens do site.

- `logo.png` — logotipo principal (recomendado PNG com fundo transparente, largura ~400px para o header)
- `hero-image.jpg` — imagem principal para a home
- `dresses/` — pasta opcional para fotos dos vestidos (ex.: `dresses/vestido-01.jpg`)

Exemplo:

public/
├─ images/
│  ├─ logo.png
│  ├─ hero-image.jpg
│  └─ dresses/
│     ├─ vestido-01.jpg
│     └─ vestido-02.jpg
